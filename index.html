Flexbox Layout


The Flexible Box Layout Module makes it easier to design flexible responsive layouts structures

With Flexbox there is no need to use float or positioning.

With media queries this can be extremely useful for building layouts that change for different screen sizes and devices



Flexbox Elements

The Flexbox is composed of 2 elements:

  • The Flex Container
  • The Flex Items

Flex Container

The flex container is RESPONSIBLE for CONTAINING the flex items.

It is the parent element.

The flex container becomes flexible by setting the display property as "flex" ( display: flex; ).

Flex Container Properties

display

Specifies the display behavior, in order words, the type of rendering box for an HTML element


Most important values:

  • inline

    Displays an element as an inline element (like <span>).

    Height and width properties will have no effect.

  • block

    Displays an element as a block element (like <p>).

    The element starts on a new line and takes up the whole width

  • inline-block

    Displays an element as an inline-level block container.

    The element itself is formatted as an inline element.

    Height and width values can still be applied.

  • flex

    Displays an element as a block-level flex container

  • grid

    Displays an element as a block-level grid container

  • inline-flex

    Displays an element as an inline-level flex container

  • inline-grid

    Displays an element as an inline-level grid container

  • none

    The element is completely removed

flex-direction

Defines in which direction the container will stack the flex items

Accepted values:

  • column

    Stacks the items vertically.

    From the top to the bottom.

  • column-reverse

    Stacks the items vertically.

    From the bottom to the top.

  • row

    Stacks the items horizontally.

    From the left to the right.

  • row-reverse

    Stacks the items horizontally.

    From the right to the left.

flex-wrap

Specifies whether the flex items should wrap or not.

Accepted values:

  • wrap

    Flex items will wrap if necessary.

  • nowrap

    Flex items will not wrap.

    This is set as the default value.

  • wrap-reverse

    Flexible items will wrap in reverse order.

flex-flow

Shorthand property for setting both the flex-direction and flex-wrap properties.

Example:

                                                    
    flex-flow: row wrap; ( same for "flex-direction: row" and "flex-flow: wrap" )

justify-content

Used to align the flex items on the main axis.

Accepted values:

  • center

    Aligns the flex items at the center of the container

  • flex-start

    Aligns the flex items at the beginning of the container.

    This is set as the default value.

  • flex-end

    Aligns the flex items at the end of the container.

  • space-around

    Displays the flex items with space before, between, and after the lines.

  • space-between

    Displays the flex items with space between the lines.

align-items

Used to align the flex items on the cross axis.

Accepted values:

  • center

    Aligns the flex items in the middle of the container

  • flex-start

    Aligns the flex items at the top of the container.

  • flex-end

    Aligns the flex items at the bottom of the container.

  • stretch

    Stretches the flex items to fill the container.

    This is set as the default value.

  • baseline

    Aligns the flex items such as their baselines aligns.

align-content

Used to align the flex lines.

Accepted values:

  • flex-start

    Displays the flex lines at the start of the container.

  • flex-end

    Displays the flex lines at the end of the container.

  • center

    Displays the flex lines in the middle of the container.

  • stretch

    Stretches the flex lines to take up the remaining space.

    This is set as the default value.

  • space-between

    Displays the flex lines with equal space between them

  • space-around

    Displays the flex lines with space before, between, and after them.

For perfectly centering, set "justify-content" and "align-items" to center, and the flex item will be perfectly centered

Flex Items

Flex Items Properties

The direct child elements of a flex container will automatically become flexible items

order

Specifies the order of the flex items

The first flex item does not have to appear as the first item in the layout

The value used must be a number (default is 0)

Example:

                                                    
    <div style="order: 3"></div>

flex-grow

Specifies how much a flex item will grow relative to the rest of the flex items

The value must be a number (default is 0)

Related to the speed at which the items will grow when you resize the screen

Example:

                                                    
    <div style="flex-grow: 1"></div>
    <div style="flex-grow: 1"></div>
    <div style="flex-grow: 1"></div>
    <div style="flex-grow: 8"></div> /*Will grow 8 times as fast

flex-shrink

Specifies how much a flex item will shrink relative to the rest of the flex items.

Example:

                                                    
    <div style="style="flex-shrink: 0"> 1</div> /*will not shrink

Example 2:

                                                    
    <div style="flex-shrink: 1"> 1</div>
    <div style="flex-shrink: 1"> 2</div>
    <div style="flex-shrink: 4"> 3</div> /*will shrink 4 times as fast

flex-basis

Specifies the initial length of a flex item.

Example:

                                                    
    <div style="flex-basis: 200px"></div>

flex

Shorthand property for the flex-grow, flex-shrink, and flex-basis properties.

Example:

                                                    
    <div style="flex: 0 0 200px"></div> //flex item not growable (0), not shrinkable (0), and with an initial length of 200 pixels

align-self

Specifies the alignment for the selected item inside the flexible container.

Overrides the default alignment set by the container's "align-items" property

Example:

                                                    
    <div style="align-self: center"ɯ</div>
